# DragInViewPager

```
    一个自定义控件，实现ViewPager中拖动功能，并自定义相关控制事件
```

---

#### Demo展示 直接上图
![image](./gif/ezgif-3-f03a2f1d8a.gif )

#### 一、功能介绍
1. **满足 viewpager 切换 view **
2. **非view触摸区域，触摸事件递交给底层view**
3. 联动事件处理
4. 自定义属性控制

#### 二、典型应用
1. 如地图寻宝签到，多种签到方式，在地图上面一层做相关切换逻辑操作
2. 如展示相关信息，多种展示方式，和底部视图联动
